# Just In Case

Just In Case es una aplicación web simple que te permite activar un temporizador y recibir un correo electrónico con un archivo adjunto después de un período de tiempo especificado. La aplicación también incluye un contador en tiempo real que se muestra en la interfaz de usuario.

## Características

- **Activación del temporizador:** Inicia un temporizador para enviar un correo electrónico después de ese tiempo.
- **Resetear temporizador:** Permite reiniciar el temporizador en cualquier momento.
- **Contador en tiempo real:** Muestra el tiempo restante en tiempo real en la interfaz de usuario.

## Requisitos previos

- Node.js instalado en tu sistema.

## Instalación

1. Clona este repositorio en tu máquina local:

   ```bash
   git clone https://github.com/tu_usuario/just-in-case.git

   ```

2. Instala las dependencias:

   ```bash
   npm install

   ```

3. Configura las Variables de Entorno:

   Crea un archivo .env en la raíz del proyecto.
   Define las variables de entorno necesarias (EMAIL_USER, EMAIL_PASSWORD, DESTINATARIOS).

4. Uso
   ```bash
   npm start
   ```
